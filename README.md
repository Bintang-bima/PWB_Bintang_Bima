PWB Tugas Pak Jumadi
 1. html : Akar utama atau pembungkus yang menandakan bahwa dokumen ini adalah file HTML dari awal sampai akhir.
 2. head : Bagian yang menampung informasi teknis (meta data) tentang website, seperti judul tab dan pengaturan di balik layar.
 3. title : Menentukan nama yang muncul pada tab browser saat halaman website sedang dibuka.
 4. body : Wadah untuk semua konten utama seperti teks, gambar, dan video yang ingin diperlihatkan kepada pengunjung.
 5. h1 sampai h6 : Jenis judul (heading) dengan tingkatan berbeda; h1 untuk judul paling besar dan h6 untuk yang terkecil.
 6. hr : Membuat garis horizontal lurus yang biasanya digunakan untuk memisahkan satu bagian konten dengan bagian lainnya.
 7. p : Singkatan dari paragraf; digunakan untuk mengelompokkan teks atau tulisan biasa agar lebih rapi.
 8. b : Digunakan untuk membuat tampilan teks menjadi tebal (bold) secara visual.
 9. a : Membuat tautan (hyperlink) agar pengunjung bisa mengklik teks tersebut dan berpindah ke alamat web lain.
 10. img : Instruksi untuk memanggil dan menampilkan gambar
